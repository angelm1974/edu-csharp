# 01.3 - Typy Danych

## Cel Modułu
Zrozumienie systemów typów w C# i wybór odpowiedniego typu do każdego zadania.

## Zagadnienia
- Typy wartościowe: int, double, bool, char, decimal
- Typy referencyjne: string
- Zakresy wartości dla każdego typu
- Określanie typu zmiennej
- Overflow i underflow

## Oczekiwana Umiejętność
**Wybranie i zastosowanie odpowiedniego typu danych do zadania**

## Zadania
1. Utwórz program demonstrujący różne typy danych
2. Wypisz zakresy wartości dla int, double, bool
3. Przetestuj overflow dla int
4. Porównaj różne typy i ich zachowanie
5. Utwórz program kalkulator używający decimal

## Materiały do Przygotowania
- Przykłady dla każdego typu danych
- Tablica zakresów wartości
- Test overflow'u
