# 01.6 - Operatory

## Cel Modułu
Opanowanie operacji na danych za pomocą różnych operatorów.

## Zagadnienia
- Operatory arytmetyczne: +, -, *, /, %
- Operatory porównania: ==, !=, <, >, <=, >=
- Operatory logiczne: &&, ||, !
- Operatory przypisania: =, +=, -=, *=, /=, %=
- Operatory inkrementacji/dekrementacji: ++, --
- Priorytet operatorów

## Oczekiwana Umiejętność
**Wykonywanie i kombinowanie operacji na danych**

## Zadania
1. Utwórz program kalkulator z operatorami arytmetycznymi
2. Przetestuj operatory porównania
3. Utwórz program z logicznymi wyrażeniami warunkowymi
4. Porównaj różne operatory przypisania
5. Przetestuj priorytet operatorów

## Materiały do Przygotowania
- Tabela operatorów z priorytetami
- Przykłady dla każdej grupy operatorów
- Program kalkulator
- Program z warunkami logicznymi
