# 01.4 - Zmienne, Stałe, Literały

## Cel Modułu
Nauczenie się zarządzania danymi w programie poprzez zmienne i stałe.

## Zagadnienia
- Deklaracja zmiennych (var, explicit type)
- Inicjalizacja zmiennych
- Zakresy zmiennych (scope)
- Stałe (const)
- Pola readonly
- Nazewnictwo zmiennych

## Oczekiwana Umiejętność
**Efektywne zarządzanie danymi w programie**

## Zadania
1. Utwórz program z różnymi typami zmiennych
2. Zdeklaruj i zainicjalizuj zmienne w różnych zakresach
3. Przetestuj błędy scope'u
4. Porównaj const vs readonly
5. Utwórz program który używa zmiennych do obliczeń
