# 01.5 - Konwersja Typów

## Cel Modułu
Nauka bezpiecznej zamiany typów danych w różnych scenariuszach.

## Zagadnienia
- Konwersja niejawna (implicit conversion)
- Konwersja jawna (explicit/casting)
- Metoda Parse()
- Metoda TryParse()
- Klasa Convert
- Obsługa błędów konwersji

## Oczekiwana Umiejętność
**Bezpieczna zamiana typów danych z obsługą błędów**

## Zadania
1. Utwórz program demonstrujący konwersje niejawne
2. Przetestuj konwersje jawne i overflow
3. Użyj Parse() do konwersji string'a na liczbę
4. Zastosuj TryParse() do bezpiecznej konwersji
5. Porównaj różne metody konwersji

## Materiały do Przygotowania
- Przykłady konwersji niejawnych
- Przykłady konwersji jawnych z casting'iem
- Program demonstrujący Parse vs TryParse
- Program z obsługą błędów
