# 01.2 - Składnia i Struktura Programu

## Cel Modułu
Nauka prawidłowej składni C# i konwencji kodowania.

## Zagadnienia
- Konwencje nazewnictwa (PascalCase, camelCase)
- Średniki (;) - koniec instrukcji
- Nawiasy klamrowe {} - zakresy bloków kodu
- Komentarze jednoliniowe (//)
- Komentarze wieloliniowe (/* */)
- Indentacja i formatowanie kodu

## Oczekiwana Umiejętność
**Czytanie i pisanie prawidłowego, czytelnego kodu C#**

## Zadania
1. Przeanalizuj strukturę istniejącego programu (PierwszyKod.cs)
2. Dodaj komentarze do kodu
3. Zastosuj prawidłowe konwencje nazewnictwa
4. Popraw indentację kodu
5. Napraw składnię (dodaj brakujące średniki)

## Pliki
- `PierwszyKod/Program.cs` - Przykład kodu do analizy
